## 1. 渣女识别

### 1.1 渣女常见类型

| 类型 | 特征 | 识别方法 |
|------|------|----------|
| 养鱼型 | 同时和多人保持暧昧 | 从不公开你们的关系，朋友圈从不发你 |
| 备胎型 | 把你当情绪垃圾桶 | 只在需要时找你，不需要时就消失 |
| 索取型 | 不断要求礼物/金钱 | 经常暗示你要买什么东西 |
| 海王型 | 到处撩，不专一 | 与多人同时聊天，回复速度异常快 |
| 备胎转正型 | 先养备胎后甩你 | 一旦找到更好的，立刻消失 |

### 1.2 渣女行为模式

```markdown
## 渣女特征检查表

- [ ] 她从不主动公开你们的关系
- [ ] 她的朋友圈从来没有你的痕迹
- [ ] 她只在需要你的时候才联系你
- [ ] 她经常消失几天不回复
- [ ] 她从不让你见到她的朋友或家人
- [ ] 她经常暗示你要花钱
- [ ] 她同时和多人保持暧昧
- [ ] 她忽冷忽热，让你摸不清
- [ ] 她从不投入，只想索取
- [ ] 她总说自己很忙，没时间见面
```

### 1.3 渣女话术

| 话术 | 含义 |
|------|------|
| "我现在不想谈恋爱" | 你是备胎，不需要负责 |
| "我们还是做朋友吧" | 继续索取但不给你名分 |
| "我前男友伤害过我" | 为自己的渣找借口 |
| "你不够爱我" | 不断要求你付出更多 |
| "我很忙" | 忙着和其他人聊天 |

### 1.4 应对策略

```
识别到渣女后的策略：

1. 立即止损
   - 不要继续投入时间/金钱
   - 减少联系频率

2. 保护自己
   - 不要透露财务信息
   - 不要给任何钱

3. 优雅退出
   - 不要吵架
   - 保持距离
```

## 2. 酒托/饭托识别

### 2.1 什么是酒托/饭托

通过社交软件搭讪男性，以约会为名带到特定酒吧/餐厅进行高额消费的女性。她们通常受雇于店家，拿提成。

### 2.2 酒托特征

| 特征 | 说明 |
|------|------|
| 相貌出众 | 照片通常是美女 |
| 主动搭讪 | 主动在社交软件上加你 |
| 快速邀约 | 聊不了几句就要求见面 |
| 指定地点 | 指定去某个酒吧/餐厅 |
| 消费极高 | 酒水/菜品价格离谱 |

### 2.3 酒托话术

| 话术 | 危险信号 |
|------|----------|
| "我知道一家很棒的酒吧，一起去吧" | 指定地点，可能与店家合作 |
| "我请客" | 到店后找借口让你买单 |
| "我朋友在那里上班" | 可能拿提成 |
| "就喝一杯" | 到店后会有各种追加消费 |
| "我刚下班，好累" | 制造紧迫感让你快速决定 |

### 2.4 酒托识别检查表

```markdown
## 酒托风险检查

- [ ] 她主动加你并搭讪
- [ ] 她快速要求见面（少于3天）
- [ ] 她指定具体酒吧/餐厅
- [ ] 她的照片过于精致（可能照骗）
- [ ] 她不提任何关于自己工作/生活的话题
- [ ] 她拒绝换地方或AA
- [ ] 见面后她马上点最贵的酒

高风险项：>=3项 → 酒托概率高
```

### 2.5 应对策略

```
遇到疑似酒托时的策略：

1. 提前侦查
   - 在大众点评查看该店评价
   - 搜索"酒托"关键词

2. 坚持换地方
   - "我常去另一家" → 看她反应
   - 如果她坚持 → 酒托概率高

3. 坚持AA或换地方
   - 如果消费后她坚持让你买单 → 酒托
   - 果断找借口离开

4. 保护财务
   - 不要带太多现金
   - 提前设置消费限额
```

## 3. 军人诈骗识别

### 3.1 什么是军人诈骗

骗子伪装成军人身份，通过社交软件认识受害者，建立感情后以各种理由骗钱。常见于跨国军人恋爱诈骗。

### 3.2 军人诈骗特征

| 特征 | 说明 |
|------|------|
| 外籍军人 | 通常是美军/维和士兵 |
| 突然搭讪 | 主动在社交软件上认识你 |
| 完美人设 | 长相帅气，性格温柔，有爱心 |
| 远离家乡 | 声称在海外执行任务 |
| 家人问题 | 经常提起家人但无法视频 |

### 3.3 军人诈骗话术

| 话术 | 危险信号 |
|------|----------|
| "我在海外执行秘密任务" | 制造神秘感，无法验证 |
| "我的家人遇到紧急情况" | 骗钱的借口 |
| "我受伤了，需要钱治疗" | 冒充伤病骗钱 |
| "我会寄给你包裹/钱" | 骗取海关费/手续费 |
| "我退休后去找你" | 画大饼，长期培养感情 |

### 3.4 识别检查

```markdown
## 军人诈骗风险检查

- [ ] 外籍军人主动搭讪你
- [ ] 无法视频通话（各种借口）
- [ ] 从不语音或视频
- [ ] 照片看起来像网络图片
- [ ] 关系发展极快（几天就表白）
- [ ] 开始用各种理由要钱
- [ ] 从不提具体军队驻地信息

高风险项：>=3项 → 军人诈骗概率高
```

### 3.5 应对策略

```
遇到疑似军人诈骗时的策略：

1. 验证身份
   - 要求视频通话（真军人可以有）
   - 询问具体驻地信息
   - 搜索他的照片（反图搜索）

2. 财务红线
   - 任何金钱要求立即拒绝
   - 不要透露银行账户信息
   - 不要帮他转账（可能是洗钱）

3. 终止联系
   - 一旦发现是诈骗，立即拉黑
   - 不要恋战，不要试图"拯救"他
```

## 4. 创业/成功人士诈骗识别

### 4.1 什么是创业诈骗

骗子伪装成成功创业者、投资人、企业家，通过社交软件认识受害者，以高回报投资或借钱为由骗钱。

### 4.2 创业诈骗特征

| 特征 | 说明 |
|------|------|
| 人设完美 | 有钱、有事业、有品位 |
| 展示财富 | 朋友圈大量豪车、名表、出入高端场所 |
| 快速建立感情 | 很快表白，说你是他的"真命天女" |
| 投资机会 | 给你一个"稳赚不赔"的投资机会 |
| 借钱周转 | 声称资金周转不灵，借钱应急 |

### 4.3 创业诈骗话术

| 话术 | 危险信号 |
|------|----------|
| "我有一个项目，年化30%收益" | 高回报骗局 |
| "我的资金周转有点问题" | 借钱骗局的开始 |
| "你帮我转账，我给你提成" | 可能是洗钱 |
| "我带你一起投资" | 诱导你投入资金 |
| "我对你很认真，想和你结婚" | 长期培养感情后再骗 |

### 4.4 识别检查

```markdown
## 创业诈骗风险检查

- [ ] 朋友圈过度展示财富
- [ ] 主动和你建立感情
- [ ] 很快提出投资机会
- [ ] 要求你投钱或转账
- [ ] 承诺高回报
- [ ] 不愿意线下见面或只是短暂见面
- [ ] 身份难以核实

高风险项：>=3项 → 创业诈骗概率高
```

### 4.5 应对策略

```
遇到疑似创业诈骗时的策略：

1. 冷静分析
   - 高回报必然伴随高风险
   - 没有"稳赚不赔"的投资

2. 核实身份
   - 搜索公司信息（天眼查）
   - 核实投资项目
   - 不要相信"内部消息"

3. 财务红线
   - 不投钱
   - 不转账
   - 不帮转账（可能洗钱）

4. 终止联系
   - 发现是诈骗立即拉黑
```

## 5. 杀猪盘识别

### 5.1 什么是杀猪盘

"杀猪盘"是近年来高发的诈骗方式。骗子通过社交软件认识受害者，建立深度情感关系（"养猪"），然后骗取大量金钱（"杀猪"）。

### 5.2 杀猪盘特征

| 特征 | 说明 |
|------|------|
| 完美人设 | 高、富、帅、专一 |
| 完美相遇 | 就像命中注定 |
| 快速升温 | 很快确定恋爱关系 |
| 异地/海外 | 经常不在同一个城市 |
| 投资背景 | 发现"赚钱机会" |

### 5.3 杀猪盘流程

```
1. 养猪阶段（1-3个月）
   - 每天问候，关心备至
   - 甜言蜜语，情感操控
   - 建立深度依赖

2. 杀猪阶段
   - 发现"投资机会"
   - 带你赚钱
   - 先让你小赚
   - 诱导大额投入
   - 然后消失
```

### 5.4 杀猪盘话术

| 话术 | 危险信号 |
|------|----------|
| "我发现了XX平台的漏洞" | 不可信的内幕消息 |
| "我带你来赚钱" | 诱导投资 |
| "你先投一点试试" | 钓鱼 |
| "我这边有点事，需要钱周转" | 借钱骗钱 |
| "我保证让你赚钱" | 虚假承诺 |

### 5.5 识别检查

```markdown
## 杀猪盘风险检查

- [ ] 他在社交软件上完美遇见你
- [ ] 他的人设过于完美
- [ ] 他很快表白，确定关系
- [ ] 他经常在海外/异地
- [ ] 他开始透露"投资机会"
- [ ] 他让你投钱或转账
- [ ] 他给你的平台无法搜索到
- [ ] 你无法核实他的真实身份

高风险项：>=3项 → 杀猪盘概率极高
```

### 5.6 应对策略

```
遇到疑似杀猪盘时的策略：

1. 情感隔离
   - 不要被感情冲昏头脑
   - 问自己：他为什么这么好？

2. 财务红线
   - 不投钱
   - 不转账
   - 不帮转账

3. 核实身份
   - 要求视频（一定要真实视频）
   - 搜索他的照片（反图搜索）
   - 搜索他的手机号/微信号

4. 终止联系
   - 一旦涉及钱，立刻拉黑
   - 不要相信任何"投资机会"
```

## 6. 伪装色狼识别

### 6.1 什么是伪装色狼

表面看起来正经、有礼貌，实际上以骗色为目的的男性。他们通常会建立一段时间的感情，然后逐渐暴露真实目的。

### 6.2 伪装色狼特征

| 特征 | 说明 |
|------|------|
| 形象正面 | 穿着得体，谈吐文明 |
| 甜言蜜语 | 很快表达喜欢/爱 |
| 制造独处 | 经常创造两人独处的机会 |
| 试探边界 | 逐渐试探身体接触 |
| 酒后行动 | 经常约你去喝酒 |

### 6.3 伪装色狼话术

| 话术 | 危险信号 |
|------|----------|
| "我太喜欢你了"（早期） | 过快表白，建立依赖 |
| "我们私下喝点酒" | 制造机会 |
| "我们去安静的地方坐坐" | 制造独处 |
| "你都25了还没经验？" | 贬低+诱导 |
| "你不想吗？" | 施压进行性行为 |

### 6.4 识别检查

```markdown
## 伪装色狼风险检查

- [ ] 很快表白，说爱你
- [ ] 经常创造独处机会
- [ ] 经常约你喝酒
- [ ] 喜欢试探身体接触边界
- [ ] 在酒后试图进行更亲密的接触
- [ ] 不尊重你说"不"的权利
- [ ] 用各种方式给你压力

高风险项：>=3项 → 伪装色狼概率高
```

### 6.5 应对策略

```
遇到疑似伪装色狼时的策略：

1. 保护自己
   - 第一次见面选择公共场所
   - 不要轻易喝酒
   - 告诉朋友你的位置

2. 观察行为一致性
   - 他说的和做的是否一致
   - 他是否尊重你的边界
   - 他是否急于进展

3. 直接沟通
   - 明确表达你的边界
   - 如果他越界，立即离开

4. 果断退出
   - 一旦发现不对，立即拉黑
   - 不要恋战，不要给机会
```

## 7. 综合防御策略

### 7.1 首次见面安全清单

```
✅ 第一次见面选择公共场所（咖啡厅、餐厅、商场）
✅ 告诉朋友你的位置
✅ 不要喝酒或少量
✅ 自己付自己的账单
✅ 注意对方行为的一致性
✅ 如果感觉不对，立即离开
```

### 7.2 财务红线

```
❌ 不给认识不久的人借钱
❌ 不投任何"高回报"投资
❌ 不帮任何人转账（可能洗钱）
❌ 不透露银行账户信息
❌ 不点击对方发的任何链接
```

### 7.3 身份验证清单

```
✅ 要求视频通话（必须真实视频）
✅ 搜索手机号/微信号
✅ 搜索照片（反图搜索）
✅ 搜索公司/职业信息
✅ 不要相信单方面的"证据"
```

### 7.4 信任评估体系

| 分数 | 标准 | 行动 |
|------|------|------|
| 0-30 | 高风险 | 立即止损 |
| 30-60 | 中风险 | 谨慎投入 |
| 60-80 | 低风险 | 正常交往 |
| 80+ | 较可信 | 继续发展 |

**评估维度**：
- 身份可核实程度（0-20）
- 行为一致性（0-20）
- 金钱往来（0-20）
- 关系投入对等（0-20）
- 社交圈融合（0-20）

## 附录：紧急情况处理

### 如果已经被骗

1. **保留证据**
   - 保存所有聊天记录
   - 保存转账记录
   - 保存照片/视频

2. **报警**
   - 立即拨打110
   - 到当地派出所报案
   - 提供所有证据

3. **止损**
   - 停止继续转账
   - 冻结相关账户

### 如果疑似诈骗

1. **不要继续投入**
   - 停止金钱往来
   - 停止感情投入

2. **收集证据**
   - 截图保存聊天记录
   - 保存转账凭证

3. **举报**
   - 在社交平台举报
   - 拨打12377举报
