# Love Simulation Skill

基于 NLP、荣格心理学和博弈论的恋爱模拟助手，帮助用户练习社交软件上的追求对话，提升共情能力、掌握潜意识植入技术、识别 PUA 套路和骗色骗财行为。

## 功能特性

- **双模式选择**：健康沟通模式 / 防身训练模式 / 安全防护模式
- **多平台模拟**：微信、Tinder、探探等
- **心理学驱动**：NLP 米尔顿模型、荣格原型系统、博弈论分析
- **六维评分体系**：安全边界、情绪共情、价值展示、对话推进等
- **诈骗识别训练**：酒托、杀猪盘、军人诈骗、创业诈骗等

## 使用方式

在支持 skills 的 AI 助手中：
- 输入 `/love` 或 `/love-sim`
- 描述场景如"模拟微信聊天"、"练习追女生"

## 核心原则

1. **真诚优先**：所有技巧必须建立在尊重对方的基础上
2. **反操控**：明确标注并禁止 PUA 红色技巧
3. **可量化**：每次对话后给出多维评分和具体改进建议
4. **心理学底层**：理解 NLP、荣格、博弈论的底层逻辑
